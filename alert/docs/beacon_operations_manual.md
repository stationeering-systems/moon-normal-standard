# BEACON CONTROL - SYSTEM OPERATIONS MANUAL

## SYSTEM PURPOSE
Pulses the sector beacon's color and on/off state based on status provided by the Sentinel.

## FUNCTIONAL OVERVIEW
* Reads the max alert code from the sentinel
* Displays the value on the beacon: see [Beacon Signal Components](https://stationeering.substack.com/i/182428013/beacon-signal-components)

## HARDWARE INTERFACE
* **d0:** Beacon
* **d1:** Sentinel
